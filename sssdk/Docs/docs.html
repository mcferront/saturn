Renesas Technology (formerly Hitachi Semiconductor) documentation:

  • SH-1/SH-2 Programming Manual. If you want learn SuperH assembly.
  • SH7604 Hardware Manual. The chips in the Saturn. DMA controllers, watchdog timers etc.
  • SuperH RISC Engine Assembler, User's Manual
  • SH Series C Compiler, User's Manual
  • H Series Librarian, User's Manual
  • SuperH RISC Engine Simulator/Debugger, User's Manual
  • SH Series Simulator/Debugger, User's Manual (for SPARC version)

Freescale (formerly Motorola) documentation:

  • Motorola M68000 Family Programmer's Reference Manual. For the sound CPU, also useful for Megadrive people.

Psy-Q documentation:

  • Systems Manual. Contains the docs for ASM68K/ASMSH which have an almost identical syntax to SNASM68K and SNASMSH2.
  • Assorted Psy-Q tools. Most related to their development hardware, but there's some good stuff there like the SCU DSP emulator.

Cross Products documentation:

  • Mirage CD Emulator Manual. Also contains a reference for their CD image building tool which should be compatible with Sega's.

Compilers, libs and tools:

  • Saturn compiler (4,8MB) (Outdated, Cygnus v.2.7. I recommend getting the KPIT package, see further down.)
  • SGL libraries (5,7MB)
  • SGL tweaked for Unix-based development (2MB). The fixing was done by ExCyber.
  • Sega Basic Library 6.01 is a combination of the low-level libs of earlier SBL versions and the high-level SGL library. Comes with sources!
  • bgCon, a tool for converting background graphics.
  • HSITOOL is a COFF downloader for the official development hardware. Comes with sources.
  • Tools for building an IP.BIN for the GNU development environment.
  • Virtual CD software. Includes tools for building disc images.
  • SS-SDK Win95 Graphic Tools, ver. 1.0J.
  • CF seems to be a file converter for binaries.

You can get a free PC extractor for .sit files from StuffIt. However, it seems to have problems with some of the archives here.
  • ADPCM Encoder.

  • Sega graphics tools, June 1996 distribution. Also includes two PC utilities.
  • Sound tools.
  • Sound demos.

Links:

  • KPIT's GNUSH is a complete up-to-date free cross-compiling GNU toolchain configured for SuperH processors. Precompiled versions are available for both Windows and Linux. Get the COFF version for compatibility with SEGA's libraries.
  • ASH WARE offer a precompiled 68K/Coldfire toolchain, based on EGCS 1.1.2 which is fairly old, but it's configured for COFF and might be useful.
  • MotoRobots have a slightly newer 68K/Coldfire toolchain, based on GCC 2.95.3 and newlib 1.9. This is an ELF toolchain.
  • Bart Trzynadlowski's site has both SNASMSH2 and SNASM68K, plus other useful things.
  • Charles MacDonald's homepage, technical info on all things Sega.
